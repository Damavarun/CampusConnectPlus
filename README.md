**### CampusConnectPlus is a platform designed to manage and organize workshops and campus events efficiently.
**### It helps students and organizers stay updated with upcoming sessions, registrations, schedules, and event information.

*### SEPM (SOFTWARE ENGINEERING AND PROJECT MANAGEMENT)**
**

### 🚀Features

📅 Workshop & Event Scheduling

📝 Event Registration Management

📢 Announcements & Notifications

📊 Participant Tracking

📂 Resource Sharing for Workshops


### Purpose

The main objective of this project is to simplify campus activity management by providing a centralized system for:

Organizing technical workshops

Managing college events

Tracking student participation

Sharing workshop materials

Improving communication between organizers and students
